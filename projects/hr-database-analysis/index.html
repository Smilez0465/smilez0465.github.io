HR Database Analysis

Tools Used: MySQL, Excel

📌 Objective

To explore and analyze an HR database to answer key business questions about employee demographics, departments, and salaries.

🔍 Key Steps

  • Connected to the HR schema using MySQL
  • Queried employee gender ratios, department distributions, and salary ranges
  • Cleaned and joined tables such as employees, departments, and salaries
  • Derived insights on hiring trends and departmental composition

📈 Outcome

Presented summaries on employee demographics and potential hiring gaps. This analysis can guide HR managers in resource planning.
